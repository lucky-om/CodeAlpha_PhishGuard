PhishGuard is a high-fidelity cybersecurity awareness platform designed to educate users on the sophisticated mechanics of modern phishing attacks. By simulating real-world threat vectors—including Email, SMS (Smishing), QR Codes (Quishing), and MFA Interception—PhishGuard transforms passive learning into an interactive forensic experience.

---

## 🚀 Key Features

### 🔍 Link Analyzer
Inspect suspicious domains in a local sandbox. The engine analyzes:
- **Homograph Recognition**: Detects visually identical characters (e.g., `rn` vs `m`).
- **Typosquatting Logic**: Calculates Levenshtein distance against global brands.
- **FQDN Validation**: Strict regex enforcement to filter out random/malicious connections.

### 🧬 Forensic Phish Anatomy
Deep-dive into high-fidelity clones of major platforms like **Microsoft**, **Amazon**, and **Netflix**.
- **Interactive Hotspots**: Identify tactical deceptive markers.
- **Header Analysis**: Inspect raw technical metadata and origin signatures.

### 📱 Real-Time Attack Simulators
Experience the "Attacker Console" view of social engineering:
- **Bank OTP Interception**: See how MFA is bypassed in real-time.
- **Smishing Lab**: Analyze mobile-specific deceptive SMS patterns.
- **Quishing Lab**: Safe scanning and analysis of malicious QR patterns.
